📺 Netflix Content Analytics Dashboard
📌 Project Overview
This project presents an interactive analytics dashboard built using Power BI to analyze Netflix's content library. The goal is to uncover content trends, identify distribution patterns, and support data-driven insights into Netflix's programming strategy.

🎯 Problem Statement
Understanding Netflix's content landscape is crucial for analyzing what drives viewership and subscription engagement. This project addresses the need for clear visibility into Netflix's catalog composition across genres, regions, release dates, and content ratings.

This dashboard aims to:

Analyze content distribution across genres and types (Movies vs. TV Shows)
Identify trends in content releases over time
Understand regional content availability and preferences
Evaluate the relationship between ratings, duration, and content popularity
Assess content categorization patterns
📊 Key Metrics (KPIs)
Total Titles: Number of movies and TV shows in the catalog
Content Split: Ratio of Movies vs. TV Shows
Average Duration: Average movie length and TV show seasons
Content by Genre: Distribution across entertainment categories
Regional Coverage: Number of titles available by region
📈 Dashboard Features
1. Content Type Distribution
Visualizes the breakdown between Movies and TV Shows
Helps identify content strategy focus
2. Genre Analysis
Breaks down content by primary genre categories
Identifies which genres dominate the catalog
Reveals gaps and opportunities
3. Release Trends
Shows content additions over time
Highlights patterns in Netflix's acquisition strategy
4. Regional Availability
Maps content distribution across countries and regions
Shows which content is globally available vs. region-specific
5. Rating & Duration Insights
Analyzes content ratings (G, PG, PG-13, R, NC-17, etc.)
Compares movie durations and TV show season counts
6. Interactive Filters
Genre
Content Type (Movie/TV Show)
Release Year
Rating
Region
These allow users to dynamically explore the Netflix catalog.

🧠 Key Insights
Movies dominate Netflix's catalog, but TV Shows may drive more engagement
Certain genres are significantly overrepresented compared to others
Most content is released in recent years, with older content strategically curated
Content availability varies significantly by region
Popular ratings (PG-13, TV-14) represent a large portion of the library
⚠️ Limitations
The dataset reflects catalog composition at a point in time and does not include viewing metrics or subscriber data
Cannot determine actual viewer engagement or popularity
Regional data may not reflect real-time availability
Does not include financial data or licensing costs
